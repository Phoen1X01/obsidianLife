==JS== - язык программирования (нужен для логики, бекенда и иногда анимаций)

В JS есть 3 способа создания переменной:
- const name = 1; менять значение константы нельзя
- var name2 = 1; глобальная переменная, вне зависимости от вложенности/расположения, его лучше не использовать короче
- len name3 = 1; локальная переменная

Типы данных:
- number - это как int и floot в других языках
- string - строка
- object - JSON
- null - пустота/есть ссылка, но нет ничего
- boolean - тру фолз
- undefined - меньше чем пустота/нет ссылки
- BigInt - большей инт
